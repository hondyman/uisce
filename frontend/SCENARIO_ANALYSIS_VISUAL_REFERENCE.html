Gauge Chart Component

Good (Sharpe > 1.5)

Warning (Sharpe 0.8-1.5)

Risk (Sharpe < 0.8)

Gauge Specifications

• SVG circle path: 282.6 circumference
• Value displayed in center
• Sizes: 96px (medium), 64px (small), 128px (large)
• Stroke width: 10px
• Linecap: round for smooth edges
• Rotate -90 degrees to start at top

Responsive Design Breakpoints

Breakpoint Width Layout Change
mobile < 640px Stack: Left 100% + Right 100%, single column
tablet 640px - 1024px 40% / 60% split, adjusted spacing
desktop > 1024px 33% / 67% split (default), full features

User Interaction Flow

User selects Portfolio ↓ User selects Scenario ↓ User clicks "Run Analysis" ↓ POST /api/portfolio/:id/scenario ↓ Backend executes ScenarioAnalysis workflow ↓ Returns AnalysisResult object ↓ Display results in right panel ↓ Add to Analysis History ↓ User can click history item to reload
