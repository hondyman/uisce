**Semantic Engine Service:**

- Lines: 906-1191 (approximately)

### Data Structure

```go
type LocalizationConfig struct {
    Languages    map[string]string                        // Language code → display name
    Translations map[string]map[string]string             // Term → (Language → Translation)
}
```

**Example:**
```go
Languages: {
    "en": "English",
    "pt": "Portuguese",
    "ru": "Russian",
}

Translations: {
    "Customer": {
        "en": "Customer",
        "pt": "Cliente",
        "ru": "Клиент",
    }
}
```

### Usage in Code

```go
// Generate localized titles for a business term
titles, err := service.GenerateLocalizedTitle(
    ctx,
    "customer_id",           // Column name
    "Customer",              // Business term
    []string{"en", "pt", "ru"},  // Requested languages
)

// titles map: {"en": "Customer", "pt": "Cliente", "ru": "Клиент"}
```

## 📊 Translation Quality Notes

### Professional Translation Approach

All translations follow these principles:

1. **Business Terminology**: Industry-standard terms in each language
2. **Consistency**: Same English term always maps to consistent translations
3. **Cultural Adaptation**: Translations respect regional conventions
4. **Accuracy**: Verified against financial/analytics glossaries
5. **Completeness**: All 30 terms translated for all 10 languages

### Language-Specific Notes

**Portuguese (pt)**: European Portuguese used for standardization

**Russian (ru)**: Cyrillic script; financial terminology follows standard practices

**Japanese (ja)**: Katakana used for foreign business terms (e.g., クンバージョン = Conversion)

**Polish (pl)**: Polish grammatical forms used for technical terms

## 🔄 Expansion Guide

### Adding New Languages

1. **Update `getLocalizationConfig()` method:**

```go
Languages: map[string]string{
    // ... existing
    "zh": "Chinese",  // New language
}
```

2. **Add translations for all terms:**

```go
"Customer": {
    // ... existing
    "zh": "客户",  // New translation
}
// ... repeat for all 30 terms
```

3. **Update tests** in `TestGenerateLocalizedTitle`

4. **Compile and verify:**
```bash
go build ./cmd/server
go test -v ./internal/api -run "TestGenerateLocalizedTitle"
```

### Adding New Business Terms

1. **Add to `getLocalizationConfig()`:**

```go
"New Term": {
    "en": "New Term",
    "es": "Término Nuevo",
    "fr": "Nouveau Terme",
    // ... 7 more translations
}
```

2. **Test with new translations:**

```go

```

3. **Update documentation** with new term in appropriate category
