# Google Gemini Integration - Quick Start

## 5-Minute Setup

### Step 1: Get API Key (1 minute)

1. Visit: https://makersuite.google.com/app/apikey
2. Click "Create API Key"
3. Copy the key (looks like: `AIzaSyD...`)

### Step 2: Set Environment Variable (1 minute)

```bash
export GEMINI_API_KEY="AIzaSyD..."
```

Verify:
```bash
echo $GEMINI_API_KEY  # Should print your key
```

### Step 3: Initialize Service (1 minute)

**Backend:**
```go
package main

import (
    "os"
    "log"
    "github.com/hondyman/semlayer/backend/internal/analytics"
)

func main() {
    service := &analytics.SemanticMappingService{}
    
    apiKey := os.Getenv("GEMINI_API_KEY")
    if err := service.InitializeGeminiProvider(apiKey); err != nil {
        log.Fatal(err)
    }
    
    // Done! AI titles now enabled
}
```

**Semantic-Engine:**
```go
package main

import (
    "os"
    "log"
    "github.com/hondyman/semlayer/services/semantic-engine/internal/services"
)

func main() {
    service := &services.SemanticMappingService{}
    
    apiKey := os.Getenv("GEMINI_API_KEY")
    if err := service.InitializeGeminiProvider(apiKey); err != nil {
        log.Fatal(err)
    }
    
    // Done! AI titles now enabled
}
```

### Step 4: Test It (2 minutes)

```go
import "context"

ctx := context.Background()

title, confidence, err := service.GenerateAITitle(
    ctx,
    "cust_acq_cost",
    map[string]interface{}{"data_type": "decimal"},
    "decimal",
)

if err != nil {
    log.Printf("Error: %v", err)
} else {
    log.Printf("✅ Generated: '%s' (confidence: %.2f)", title, confidence)
}
```

Expected output:
```
✅ Generated: 'Customer Acquisition Cost' (confidence: 0.95)
```

```go
// Instead of: "cust_acq_cost"
// Get: "Customer Acquisition Cost"

properties := service.inferSemanticTermProperties(column, "MEASURE", "cust_acq_cost")
```

```go
for _, column := range databaseColumns {
    title, confidence, _ := service.GenerateAITitle(ctx, column.Name, column.Metadata, column.DataType)
    glossaryEntry := GlossaryEntry{
        ColumnName: column.Name,
        Title:      title,
        Confidence: confidence,
        Source:     "Gemini",
    }
    glossary.Add(glossaryEntry)
}
```

## Troubleshooting

### "LLM provider not configured"

```go
// Check if initialized
if s.llmProvider == nil {
    // Not initialized yet
    s.InitializeGeminiProvider(apiKey)
}
```

### "API key is empty"

```bash
# Check environment variable
echo $GEMINI_API_KEY

# If empty:
export GEMINI_API_KEY="your-key-from-makersuite"

# Verify
echo $GEMINI_API_KEY
```

### "Rate limit exceeded"

```go
// Gemini free tier: 60 requests/minute
// Solution: Implement caching or wait

// Simple backoff:
import "time"

for i := 0; i < 3; i++ {
    title, _, err := service.GenerateAITitle(ctx, col, metadata, dtype)
    if err == nil {
        return title
    }
    time.Sleep(time.Second * time.Duration(1<<uint(i))) // 1s, 2s, 4s
}
```

### "Timeout error"

```go
import "context"
import "time"

// Add timeout
ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
defer cancel()

title, confidence, err := service.GenerateAITitle(ctx, columnName, metadata, dataType)
```

## Performance Tips

### 1. Batch Processing

```go
// Instead of calling one at a time:
// ❌ SLOW
for _, col := range columns {
    title, _, _ := service.GenerateAITitle(ctx, col.Name, col.Metadata, col.DataType)
}

// ✅ FAST: Process in parallel with rate limiting
import "sync"

limiter := make(chan struct{}, 10) // Max 10 concurrent
var wg sync.WaitGroup

for _, col := range columns {
    wg.Add(1)
    go func(c Column) {
        defer wg.Done()
        limiter <- struct{}{}        // Acquire
        defer func() { <-limiter }() // Release
        
        title, _, _ := service.GenerateAITitle(ctx, c.Name, c.Metadata, c.DataType)
        results[c.Name] = title
    }(col)
}
wg.Wait()
```

### 2. Caching Results

```go
type TitleCache struct {
    cache map[string]string
    mu    sync.RWMutex
}

func (tc *TitleCache) GetOrGenerate(ctx context.Context, colName string) (string, error) {
    tc.mu.RLock()
    if title, exists := tc.cache[colName]; exists {
        tc.mu.RUnlock()
        return title, nil
    }
    tc.mu.RUnlock()
    
    title, _, err := service.GenerateAITitle(ctx, colName, nil, "string")
    if err != nil {
        return "", err
    }
    
    tc.mu.Lock()
    tc.cache[colName] = title
    tc.mu.Unlock()
    
    return title, nil
}
```

### 3. Fallback Strategy

```go
title, confidence, err := service.GenerateAITitle(ctx, columnName, metadata, dataType)

switch {
case err != nil:
    // API failed → use rule-based
    title = service.generateBusinessTitle(columnName, "DIMENSION")
    
case confidence < 0.8:
    // Low confidence → use rule-based
    title = service.generateBusinessTitle(columnName, "DIMENSION")
    
default:
    // Good AI title → use it!
}

return title
```
