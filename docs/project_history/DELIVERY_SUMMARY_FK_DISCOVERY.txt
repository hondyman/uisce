SUPPORT & TROUBLESHOOTING
================================================================================

Issue: No relationships returned

Solutions:
  1. Verify tenant_id and datasource_id are correct
  2. Check entity name matches catalog_node.node_name exactly
  3. Confirm FK edges exist: SELECT COUNT(*) FROM catalog_edge 
     WHERE relationship_type='foreign_key'
  4. Check logs: tail -50 backend/backend.log

Issue: Backend won't start

Solutions:
  1. go clean -cache
  2. go mod tidy
  3. go build -o semlayer-backend
  4. ./semlayer-backend

Issue: 500 error from API

Solutions:
  1. Check backend logs for error details
  2. Verify database connection is working
  3. Check parameter format (UUID, not string)
  4. Restart backend service
