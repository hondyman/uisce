SUPPORT & TROUBLESHOOTING
================================================================================

Q: How do I start?
A: Follow TEMPORAL_QUICK_START.md (5 minutes)

Q: Search Attributes not working?
A: Run: curl http://localhost:8080/api/temporal/setup-cli-script | bash

Q: API endpoints returning 404?
A: Verify RegisterTemporalAdminRoutes() called in api.go

Q: Grafana empty?
A: Check http://localhost:9090/targets to verify temporal metrics scraping

Q: Dashboard not visible?
A: Check /temporal-admin route registered in AppRoutes.tsx

Q: Multi-tenant issues?
A: Ensure X-Tenant-ID header sent in API calls (see agents.md)
