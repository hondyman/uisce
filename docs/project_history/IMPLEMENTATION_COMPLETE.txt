DATA FLOW ARCHITECTURE
======================
Entity Schema (API)
    ↓
entitySchema prop (passed to component)
    ↓
getFieldsForEntity() + getFieldsForSubtype()
    ↓
getAllAvailableFields() (merged list)
    ↓
Field dropdown (populated)
    ↓
Selected field type detected
    ↓
getOperatorsForFieldType()
    ↓
Operator dropdown (filtered)
    ↓
Value input (type-specific)

CONDITION DATA FORMAT
====================
Input (to backend):
{
  "field": "company_name",
  "operator": "contains",
  "value": "ABC"
}

Extended (in UI state):
{
  "field": "company_name",
  "fieldType": "text",        ← UI metadata
  "operator": "contains",
  "value": "ABC",
  "fieldLabel": "Company Name" ← UI metadata
}
