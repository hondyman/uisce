## 📍 Category Locations

### 🔵 CATALOG (Blue #2196F3)
**Purpose:** Data discovery & organization

**What's inside:**
- API Catalog, Schema Explorer, Views Catalog
- Business Glossary, Catalog Setup, Data Domains

**When to use:**
- Browse available data
- Understand data structure
- Manage semantic terms

---

### 🟣 WEAVE (Purple #9C27B0)
**Purpose:** Semantic layer management

**What's inside:**
- Bundles, Models, Calculations
- Semantic Mapper, Lineage, Drift Reports
- Policies, Roles, Access Control

**When to use:**
- Build semantic packages
- Create data models
- Manage governance & policies

---

### 🟠 ENTITY (Orange #FF9800)
**Purpose:** Business operations

**What's inside:**
- Entity Manager, Business Processes
- Tenants, Configuration, Queries
- Analytics, Reports, Upgrades

**When to use:**
- Manage entities & relationships
- Design business processes
- Monitor system performance
