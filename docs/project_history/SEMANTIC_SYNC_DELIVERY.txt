## 💡 How It Works

```
User Creates Metric in React
        ↓
Sends to Backend API
        ↓
Postgres: Insert metric_registry
        ↓
Trigger: notify_metric_registry_changed()
        ↓
NOTIFY: Sends message on channel
        ↓
Semantic Sync: Receives notification
        ↓
Generates: 3 Cube.js schemas
        ↓
Writes: ./cube-schemas/*.js
        ↓
React: Queries Cube API
        ↓
Console: Updates tables in real-time
```
