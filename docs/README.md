# Documentation Index

This directory contains all project documentation, organized by category.

## Structure

- **phases/** - Phase-specific implementation documentation
- **deliveries/** - Delivery summaries and completion reports
- **guides/** - Implementation guides and READMEs
- **references/** - Quick reference guides
- **features/** - Feature-specific documentation
  - audit/ - Audit-related docs
  - business-objects/ - Business object docs
  - entities/ - Entity management docs
  - reports/ - Reporting docs
  - semantic/ - Semantic layer docs
  - validation/ - Validation rules docs
  - workflows/ - Workflow and business process docs
- **archives/** - Archived documentation (may contain duplicates or superseded content)
- **api/** - API documentation
- **architecture/** - System architecture docs
- **deployment/** - Deployment guides
- **development/** - Development setup and guides
- **runbooks/** - Operational runbooks
- **services/** - Service-specific documentation
- **templates/** - Documentation templates
